# Dotfiles

> These are my personal configuration files. These files are not stable and are constantly changing. Overhauls happen often!

My primary operating system is MacOS Sequoia.


## Installation

Clone the dotfiles repository.

```
$ git clone https://github.com/Aresshu/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
```
Bootstrap.sh is a script to create symbolic links.

```
$ ./bootstrap.sh
Usage: bootstrap.sh [options]

   link             Create symbolic links
   backup           Create a backup of you .configs within ~/.dotfiles_backup

```
